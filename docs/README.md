# Nic's Creative Corner

## Overview
Short description of the product and goals

## Tech Stack
- Next.js (Frontend + Backend)
- MongoDB (Database)
- Mongoose (ODM)
- TanStack Query (Data Fetching)
- Clerk (Authentication)
- Stripe (Payment Handling)
- TailwindCSS (Styling)
- Zod (Validation)

## Architecture
- Monorepo structure
- Role-based access (user/admin)
- API routes handled via Next.js

## Features
- Authentication
- User dashboard
- Admin panel
- API integration
- Real-time updates

## Setup Instructions
### 1. Install dependencies
```` shell
npm install
````

### 2. Run dev server
```` shell
npm run dev
````
